# Feishu / Lark 连接器规范

用途：采集与配置身份相关的同日/本周表格行、任务记录、wiki 表格条目或交接备注。

推荐字段：

- `date`
- `title`
- `owner` 或 `assignee`
- `status`
- `link`
- `description` 或 `summary`
- `product` 或 `project`

隐私规则：

- 只使用用户授权访问。
- 如果登录或权限阻塞访问，返回“不可访问来源”的证据项。
- 不要凭记忆猜测表格内容。
- 优先保存行元数据和摘要，避免保存原始私密对话。

个人安装时，用户可以配置一个飞书表格和身份字段，让自己名字对应的记录补充本地 git 证据。
